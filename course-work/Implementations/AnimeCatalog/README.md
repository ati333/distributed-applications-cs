## Описание на проекта
"Anime Catalog" е уеб приложение, разработено на ASP.NET Core MVC 8.0. Системата позволява каталогизиране на аниме заглавия, техните автори и жанрове.
Приложението поддържа пълна функционалност за управление на съдържанието (CRUD) и защита на базата на роли.

## Функционалности
1. **База данни:** Три свързани таблици (Animes, Writers, Genres) с релации One-to-Many.
2. **Роли и Сигурност:**
   - **Гост:** Разглеждане на каталога.
   - **Потребител:** Търсене и сортиране на данни.
   - **Администратор:** Създаване, редакция и изтриване на записи.
3. **Екстри:** Възможност за добавяне на линкове към снимки (постери).

## Технически стек
- **Framework:** ASP.NET Core 8.0 MVC
- **Auth:** ASP.NET Core Identity
- **Database:** SQLite (за преносимост и лесно стартиране)
- **Frontend:** HTML5, CSS3, Bootstrap

## Инструкции за инсталация и стартиране
1. Клонирайте репозиторито или разархивирайте проекта.
2. Отворете `AnimeCatalog.sln` във Visual Studio 2022.
3. Отворете **Package Manager Console** и изпълнете командата:
   `update-database`
   *(Това ще създаде локалния файл `AnimeCatalog.db`)*.
4. Стартирайте приложението с **F5**.
